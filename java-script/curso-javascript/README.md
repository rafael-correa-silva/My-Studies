# JavaScript Studies

Repositório criado para armazenar exercícios, anotações e práticas realizadas durante meus estudos de JavaScript.

Aqui estão alguns dos primeiros projetos e testes desenvolvidos para praticar conceitos fundamentais da linguagem, manipulação de dados, interação com o usuário e lógica de programação.

## Conteúdos praticados

* Variáveis
* Tipos de dados
* Operadores
* Conversão de valores
* Interação com usuário
* Manipulação de strings
* Template strings
* Estruturas básicas do JavaScript

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo

Praticar os fundamentos do JavaScript e fortalecer a base para desenvolvimento Front-End.
