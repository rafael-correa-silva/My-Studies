# Java OOP Projects

Repositório com projetos desenvolvidos durante meus estudos de Programação Orientada a Objetos em Java.

Os projetos foram criados com foco em praticar conceitos fundamentais de POO, lógica de programação e estruturação de aplicações utilizando Java.

## Projetos

### 🥊 Fighting Simulator

Sistema de simulação de lutas entre lutadores com categorias, histórico de vitórias, derrotas e empates.

Conceitos praticados:

* Orientação a objetos
* Encapsulamento
* Relacionamento entre classes
* Regras de negócio
* Randomização

---

### 🏦 Bank Account System

Sistema simples de gerenciamento bancário com abertura de contas, depósitos, saques e pagamento de mensalidades.

Conceitos praticados:

* Encapsulamento
* Métodos getters e setters
* Regras de negócio
* Controle de saldo

---

### 🎮 Remote Control System

Simulação de um controle remoto utilizando interface e implementação de métodos.

Conceitos praticados:

* Interfaces
* Abstração
* Encapsulamento
* Controle de estados

---

### 📱 Cellphone System

Projeto simples criado para praticar atributos, métodos e estruturação de classes em Java.

## Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)

## Objetivo

Praticar os principais conceitos de desenvolvimento orientado a objetos utilizando Java através de pequenos sistemas e simulações.
