# Student Management System

Sistema desktop desenvolvido em Java utilizando Swing para gerenciamento de alunos, disciplinas e cálculo de médias.

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO), interfaces gráficas com Java Swing e manipulação de eventos.

A aplicação permite cadastrar alunos, adicionar disciplinas com notas e calcular médias de forma simples e intuitiva através de uma interface moderna e personalizada.

## 🚀 Tecnologias utilizadas

* Java
* Java Swing
* Programação Orientada a Objetos (POO)

## 💻 Funcionalidades

* Cadastro de alunos
* Remoção de alunos
* Cadastro de disciplinas
* Cálculo automático de médias
* Interface gráfica moderna
* Botões personalizados
* Layout organizado e responsivo

## 🎯 Objetivo

Aplicar na prática conceitos fundamentais de desenvolvimento desktop com Java, incluindo:

* Estruturas de dados
* Eventos
* Interfaces gráficas
* Classes e objetos
* Encapsulamento
* Organização de código
